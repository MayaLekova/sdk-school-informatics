# sdk-school-informatics

Това хранилище съдържа материали за основен курс от След-дипломната квалификация на ФМИ, СУ "Св. Климент Охридски", Училищен курс по информатика. Тъй като обучаемите са този курс са 8-класници, където е възможно ще се стараем да предоставяме връзки към материали и авторски такива на български език.

Материалите включват теоретични елементи, алгоритми, представени чрез псевдокод и кратки примери на Python.

This repository contains materials for teachers learning the "School informatics" course for teaching 8-th grade students the basis of informatics.

## Теми по дни

### Ден 1

- данни и съхранение на информацията
- числата и тяхното представяне
- упражнение:
  - задачи за бройни (числови) системи
  - преобразуване на числа от 10-тична в 2-ична бройна система
  - задача за разминаване на кораби

--------------

### Ден 2

- понятие за позиционна бройна система
- алгоритми за преобразуване на числата от една бройна система в друга

--------------

### Ден 3

- общи понятия от архитектурата на компютъра
- прости алгоритми - сортиране, линейно търсене
- упражнение: "имплементиране" на алгоритмите по два начина:
  - чрез блок-схема
  - чрез псевдо-код

- въведение в езика Python

--------------

### Ден 4

- по-сложни алгоритми - двоично търсене, сортиране чрез сливане
- визуализация на алгоритми
- упражнение: имплементиране на алгоритми чрез Python

--------------
