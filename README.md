# sdk-school-informatics

Това хранилище съдържа материали за основен курс от След-дипломната квалификация на ФМИ, СУ "Св. Климент Охридски", Училищен курс по информатика. Тъй като обучаемите са този курс са 8-класници, където е възможно ще се стараем да предоставяме връзки към материали и авторски такива на български език.

Материалите включват теоретични елементи, алгоритми, представени чрез псевдокод и кратки примери на Python.

This repository contains materials for teachers learning the "School informatics" course for teaching 8-th grade students the basis of informatics.
